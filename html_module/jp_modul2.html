ℹ️ 3 Informationen

📖 3.1 Verknüpfung von JavaScript und HTML

Um eine JavaScript-Datei mit einer HTML-Datei zu verknüpfen, kannst du das <script>-Tag verwenden. Beispiel:

            
                <script src="script.js"></script>

Das src-Attribut gibt den Pfad zur JavaScript-Datei an. Du kannst das <script>-Tag im <head>- oder im <body>-Bereich deiner HTML-Datei platzieren.

Hier ist ein Beispiel für eine einfache HTML-Datei mit einer verknüpften JavaScript-Datei:

            
                <!DOCTYPE html>
                <html lang="de">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Mein Projekt</title>
                    <script src="script.js"></script>
                </head>
                <body>
                    <h1>Willkommen zu meinem Projekt</h1>
                </body>
                </html>

In diesem Beispiel wird die JavaScript-Datei script.js in die HTML-Datei eingebunden.

Du kannst auch Inline-JavaScript verwenden, indem du den JavaScript-Code direkt im <script>-Tag schreibst:

            
                <script>
                    console.log("Hallo Welt!");
                </script>

Dieser Code gibt "Hallo Welt!" in der Konsole aus, wenn die Seite geladen wird.

Es ist jedoch empfehlenswert, JavaScript in separaten Dateien zu organisieren, um den Code übersichtlicher zu gestalten.

Hier ist ein Beispiel für eine einfache JavaScript-Datei:

            
                // script.js
                console.log("Hallo Welt!");

Diese Datei enthält den gleichen Code wie das Inline-JavaScript-Beispiel, aber in einer separaten Datei.

Zusammenfassend kannst du JavaScript in HTML-Dateien einbinden, indem du das <script>-Tag verwendest. Du kannst entweder Inline-JavaScript verwenden oder den Code in separaten Dateien organisieren.

Es ist wichtig, den JavaScript-Code am Ende des <body>-Tags zu platzieren, um sicherzustellen, dass die HTML-Elemente bereits geladen sind, bevor der JavaScript-Code ausgeführt wird. Alternativ kannst du das Attribut defer verwenden, um sicherzustellen, dass der JavaScript-Code erst nach dem Laden der Seite ausgeführt wird:

            
                <script src="script.js" defer></script>

📖 3.2 Auswahl von HTML-Elementen

Mit JavaScript kannst du HTML-Elemente anhand ihrer IDs auswählen. Beispiel:

            
                let element = document.getElementById("meineID");

Das ausgewählte Element kann dann manipuliert werden.

Hier ist ein Beispiel, wie du den Text eines HTML-Elements ändern kannst:

            
                let element = document.getElementById("meinText");
                element.innerHTML = "Neuer Text!";

In diesem Beispiel wird der Text des Elements mit der ID meinText auf "Neuer Text!" geändert.

Du kannst auch andere Eigenschaften des Elements ändern, z.B. die Farbe:

            
                let element = document.getElementById("meinText");
                element.style.color = "red";

Hier wird die Textfarbe des Elements mit der ID meinText auf rot geändert.

Zusammenfassend kannst du HTML-Elemente mit JavaScript auswählen, indem du die Methode document.getElementById() verwendest. Du kannst dann den Text und andere Eigenschaften des Elements ändern.

Hier ist ein vollständiges Beispiel für eine HTML-Datei mit einem JavaScript-Skript, das den Text eines Elements ändert:

            
                <!DOCTYPE html>
                <html lang="de">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Mein Projekt</title>
                    <script src="script.js"></script>
                </head>
                <body>
                    <p id="meinText">Ursprünglicher Text</p>
                    <script>
                        let element = document.getElementById("meinText");
                        element.innerHTML = "Neuer Text!";
                    </script>
                </body>
                </html>

📖 3.3 Ereignisse und Ereignis-Listener

Ereignisse sind Aktionen, die auf einer Webseite stattfinden, z.B. ein Klick auf einen Button. Mit JavaScript kannst du auf diese Ereignisse reagieren. Beispiel:

            
                let button = document.getElementById("meinButton");
                button.addEventListener("click", function() {
                    alert("Button wurde geklickt!");
                });

Hier wird ein Ereignis-Listener hinzugefügt, der auf einen Klick auf den Button reagiert.

Du kannst auch andere Ereignisse verwenden, z.B. mouseover, mouseout, keydown, usw.

Hier ist ein Beispiel, wie du auf das mouseover-Ereignis reagieren kannst:

            
                let element = document.getElementById("meinText");
                element.addEventListener("mouseover", function() {
                    element.style.color = "blue";
                });

In diesem Beispiel wird die Textfarbe des Elements mit der ID meinText auf blau geändert, wenn der Mauszeiger darüber fährt.

Zusammenfassend kannst du mit JavaScript auf Ereignisse reagieren, indem du Ereignis-Listener hinzufügst. Du kannst verschiedene Ereignisse verwenden, um auf Benutzerinteraktionen zu reagieren.

Hier ist ein vollständiges Beispiel für eine HTML-Datei mit einem JavaScript-Skript, das auf ein Klick-Ereignis reagiert:

            
                <!DOCTYPE html>
                <html lang="de">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Mein Projekt</title>
                    <script src="script.js"></script>
                </head>
                <body>
                    <button id="meinButton">Klick mich!</button>
                    <p id="meinText">Ursprünglicher Text</p>
                    <script>
                        let button = document.getElementById("meinButton");
                        button.addEventListener("click", function() {
                            alert("Button wurde geklickt!");
                        });
                    </script>
                </body>
                </html>

📖 3.4 jQuery

jQuery ist eine JavaScript-Bibliothek, die es einfacher macht, HTML-Elemente zu manipulieren, Ereignisse zu behandeln und Animationen zu erstellen. In den vorherigen Kapiteln hast du gelernt, wie du mit reinem JavaScript HTML-Elemente auswählst, deren Eigenschaften änderst und auf Ereignisse reagierst. Mit jQuery kannst du diese Aufgaben mit weniger Code umsetzen.

🔹 Verknüpfung von jQuery

Um jQuery zu verwenden, musst du die jQuery-Bibliothek in dein Projekt einbinden. Dies kannst du tun, indem du das folgende <script>-Tag in den <head>- oder <body>-Bereich deiner HTML-Datei einfügst:

            
                <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

🔹 Auswahl von HTML-Elementen

In Kapitel 3.2 hast du gelernt, wie du HTML-Elemente mit document.getElementById() auswählst. Mit jQuery kannst du dies einfacher umsetzen:

            
                // Auswahl eines Elements mit der ID "meinElement"
                let element = $("#meinElement");

Hier wird das Element mit der ID meinElement ausgewählt. Du kannst direkt mit dem ausgewählten Element arbeiten.

🔹 Text und Eigenschaften ändern

In Kapitel 3.2 hast du gelernt, wie du den Text und die Eigenschaften eines Elements mit JavaScript änderst. Mit jQuery kannst du dies einfacher umsetzen:

            
                // Text ändern
                $("#meinElement").text("Neuer Text!");

                // Farbe ändern
                $("#meinElement").css("color", "red");

Hier wird der Text des Elements geändert und die Textfarbe auf Rot gesetzt.

🔹 Ereignis-Listener hinzufügen

In Kapitel 3.3 hast du gelernt, wie du Ereignis-Listener mit addEventListener hinzufügst. Mit jQuery kannst du dies einfacher umsetzen:

            
                // Klick-Listener hinzufügen
                $("#meinButton").click(function() {
                    alert("Button wurde geklickt!");
                });

Hier wird ein Klick-Listener zu einem Button hinzugefügt, der eine Nachricht anzeigt, wenn der Button geklickt wird.

🔹 Sichtbarkeit und Animationen

In Kapitel 3.3 hast du gelernt, wie du die Sichtbarkeit eines Elements mit style.display änderst. Mit jQuery kannst du dies einfacher umsetzen und Animationen hinzufügen:

            
                // Element ausblenden
                $("#meinElement").hide();

                // Element einblenden
                $("#meinElement").show();

                // Element mit Animation einblenden
                $("#meinElement").fadeIn(1000);

Hier wird ein Element ausgeblendet, eingeblendet und mit einer Animation versehen.

🔹 Numerische Eingaben und Berechnungen

In Kapitel 3.3 hast du gelernt, wie du numerische Eingaben aus HTML-Elementen abrufst und Berechnungen durchführst. Mit jQuery kannst du dies einfacher umsetzen:

            
                // Werte aus Eingabefeldern abrufen
                let zahl1 = parseFloat($("#zahl1").val());
                let zahl2 = parseFloat($("#zahl2").val());

                // Berechnung durchführen
                let summe = zahl1 + zahl2;

                // Ergebnis anzeigen
                $("#ergebnis").text("Summe: " + summe);

Hier werden die Werte aus zwei Eingabefeldern abgerufen, addiert und das Ergebnis in einem HTML-Element angezeigt.

Zusammenfassend kannst du mit jQuery viele der in den vorherigen Kapiteln behandelten Aufgaben einfacher und mit weniger Code umsetzen. jQuery ist besonders nützlich, wenn du dynamische Webseiten erstellen möchtest, die auf Benutzerinteraktionen reagieren.

Weitere Informationen findest du in der offiziellen jQuery-Dokumentation.

🛠️ 9 Beispiel

Hier ist ein vollständiges Beispiel, das zeigt, wie du HTML und JavaScript kombinieren kannst, um eine interaktive Webseite zu erstellen. In diesem Beispiel wird ein Button verwendet, um den Text eines HTML-Elements zu ändern und eine Nachricht in der Konsole auszugeben.

HTML-Code:

                
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>Interaktive Webseite</title>
                    </head>
                    <body>
                        <h1>Interaktive Webseite</h1>
                        <p id="textElement">Ursprünglicher Text</p>
                        <button id="changeTextButton">Text ändern</button>
                        <script src="script.js"></script>
                    </body>
                    </html>

JavaScript-Code (script.js):

                
                    // JavaScript-Code, um den Text zu ändern und eine Nachricht auszugeben
                    document.getElementById("changeTextButton").addEventListener("click", function() {
                        const textElement = document.getElementById("textElement");
                        textElement.innerHTML = "Text wurde geändert!";
                        console.log("Button wurde geklickt und der Text wurde geändert.");
                    });
