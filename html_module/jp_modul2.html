ℹ️ 3 Informationen

📖 3.1 Verknüpfung von JavaScript und HTML

Um eine JavaScript-Datei mit einer HTML-Datei zu verknüpfen, kannst du das <script>-Tag verwenden. Beispiel:

            
                <script src="script.js"></script>

Die JavaScript-Datei wird dann in die HTML-Datei eingebunden und kann verwendet werden.

📖 3.2 Auswahl von HTML-Elementen

Mit JavaScript kannst du HTML-Elemente anhand ihrer IDs auswählen. Beispiel:

            
                let element = document.getElementById("meineID");

Das ausgewählte Element kann dann manipuliert werden.

📖 3.3 Ereignisse und Ereignis-Listener

Ereignisse sind Aktionen, die auf einer Webseite stattfinden, z.B. ein Klick auf einen Button. Mit JavaScript kannst du auf diese Ereignisse reagieren. Beispiel:

            
                let button = document.getElementById("meinButton");
                button.addEventListener("click", function() {
                    alert("Button wurde geklickt!");
                });

Hier wird ein Ereignis-Listener hinzugefügt, der auf einen Klick auf den Button reagiert.

🛠️ 5 Übungsaufgaben - Praxis

🟢 Aufgabe 5.1
Erstelle ein HTML-Dokument mit einem Button. Schreibe ein Skript, das eine Nachricht in der Konsole ausgibt, wenn der Button geklickt wird.
Lösung anzeigen 
                    
                        <button id="meinButton">Klick mich!</button>
                        <script>
                            let button = document.getElementById("meinButton");
                            button.addEventListener("click", function() {
                                console.log("Button wurde geklickt!");
                            });
                        </script>
🟢 Aufgabe 5.2
Erstelle ein HTML-Dokument mit einem Textfeld und einem Button. Schreibe ein Skript, das den Inhalt des Textfelds in der Konsole ausgibt, wenn der Button geklickt wird.
🔵 Aufgabe 5.3
Erstelle ein HTML-Dokument mit einem Button. Schreibe ein Skript, das den Text eines HTML-Elements ändert, wenn der Button geklickt wird.
Lösung anzeigen 
                    
                        <button id="meinButton">Ändere Text</button>
                        <p id="meinText">Ursprünglicher Text</p>
                        <script>
                            let button = document.getElementById("meinButton");
                            let text = document.getElementById("meinText");
                            button.addEventListener("click", function() {
                                text.innerHTML = "Text wurde geändert!";
                            });
                        </script>
🔵 Aufgabe 5.4
Erstelle ein HTML-Dokument mit zwei Buttons. Schreibe ein Skript, das unterschiedliche Nachrichten in der Konsole ausgibt, je nachdem, welcher Button geklickt wird.
Lösung anzeigen 
                    
                        <button id="button1">Button 1</button>
                        <button id="button2">Button 2</button>
                        <script>
                            let button1 = document.getElementById("button1");
                            let button2 = document.getElementById("button2");
                            button1.addEventListener("click", function() {
                                console.log("Button 1 wurde geklickt!");
                            });
                            button2.addEventListener("click", function() {
                                console.log("Button 2 wurde geklickt!");
                            });
                        </script>
🔴 Aufgabe 5.5
Erstelle ein HTML-Dokument mit einem Button. Schreibe ein Skript, das den Hintergrund der Seite ändert, wenn der Button geklickt wird.
Lösung anzeigen 
                    
                        <button id="meinButton">Ändere Hintergrund</button>
                        <script>
                            let button = document.getElementById("meinButton");
                            button.addEventListener("click", function() {
                                document.body.style.backgroundColor = "lightblue";
                            });
                        </script>
🔴 Aufgabe 5.6
Erstelle ein HTML-Dokument mit einem Button. Schreibe ein Skript, das den Text eines HTML-Elements ändert und eine Nachricht in der Konsole ausgibt, wenn der Button geklickt wird.
Lösung anzeigen 
                    
                        <button id="meinButton">Ändere Text und gebe Nachricht aus</button>
                        <p id="meinText">Ursprünglicher Text</p>
                        <script>
                            let button = document.getElementById("meinButton");
                            let text = document.getElementById("meinText");
                            button.addEventListener("click", function() {
                                text.innerHTML = "Text wurde geändert!";
                                console.log("Button wurde geklickt und Text geändert!");
                            });
                        </script>
