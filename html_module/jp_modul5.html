ℹ️ 3 Informationen

📖 3.1 Zufallszahlen generieren

Mit JavaScript kannst du Zufallszahlen generieren, um interaktive Spiele zu erstellen. Beispiel:

            
                let zufallszahl = Math.floor(Math.random() * 10) + 1;
                console.log("Zufallszahl:", zufallszahl);

Hier wird eine Zufallszahl zwischen 1 und 10 generiert.

📖 3.2 Spielregeln und Scoring

Du kannst Spielregeln und Punktesysteme mit JavaScript implementieren. Beispiel:

            
                let punkte = 0;
                let antwort = prompt("Wie viel ist 2 + 2?");
                if (antwort == 4) {
                    punkte += 1;
                    console.log("Richtig! Punkte:", punkte);
                } else {
                    console.log("Falsch! Punkte:", punkte);
                }

Hier wird überprüft, ob die Antwort korrekt ist, und die Punkte werden entsprechend angepasst.

📖 3.3 Interaktive Quizfragen

Mit JavaScript kannst du interaktive Quizfragen erstellen. Beispiel:

            
                let frage = "Was ist die Hauptstadt von Deutschland?";
                let antwort = prompt(frage);
                if (antwort.toLowerCase() === "berlin") {
                    console.log("Richtig!");
                } else {
                    console.log("Falsch! Die richtige Antwort ist Berlin.");
                }

Hier wird die Benutzereingabe überprüft und eine Rückmeldung gegeben.
