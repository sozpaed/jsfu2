ℹ️ 3 Informationen

📖 3.1 Variablen

Eine Variable ist ein benannter Speicherort für Daten. Sie kann verschiedene Werte annehmen und wird mit dem Schlüsselwort let oder const deklariert.

In JavaScript gibt es zwei Arten von Variablen:

  • let: Eine Variable, deren Wert sich ändern kann.
  • const: Eine Konstante, deren Wert nicht geändert werden kann.

Beispiel:

            
                let alter = 15; // Variable
                const name = "Max"; // Konstante

Hier wird eine Variable alter mit dem Wert 15 und eine Konstante name mit dem Wert "Max" deklariert.

Variablen können auch mit dem Schlüsselwort var deklariert werden, aber es wird empfohlen, let oder const zu verwenden, da sie eine bessere Block-Sichtbarkeit bieten.

Beispiel für die Verwendung von var:

            
                var alter = 15; // Variable mit var
                console.log(alter); // Ausgabe: 15

Hier wird eine Variable alter mit dem Wert 15 deklariert und in der Konsole ausgegeben.

📖 3.2 Datentypen

JavaScript unterstützt verschiedene Datentypen, darunter:

  • string: Zeichenfolgen, z.B. "Hallo"
  • number: Zahlen, z.B. 42
  • boolean: Wahrheitswerte, z.B. true oder false
  • null: Ein leerer Wert
  • undefined: Ein nicht definierter Wert
  • object: Komplexe Datenstrukturen, z.B. Arrays oder Objekte

Beispiel:

            
                let name = "Max"; // string
                let alter = 15; // number
                let istErwachsen = false; // boolean
                let leer = null; // null
                let undefiniert; // undefined
                let person = { vorname: "Max", nachname: "Mustermann" }; // object

Hier werden verschiedene Datentypen in JavaScript verwendet. name ist ein String, alter ist eine Zahl, istErwachsen ist ein Boolean, leer ist null, undefiniert ist undefined und person ist ein Objekt.

Du kannst den Datentyp einer Variablen mit dem Operator typeof überprüfen:

            
                console.log(typeof name); // Ausgabe: string
                console.log(typeof alter); // Ausgabe: number
                console.log(typeof istErwachsen); // Ausgabe: boolean
                console.log(typeof leer); // Ausgabe: object
                console.log(typeof undefiniert); // Ausgabe: undefined
                console.log(typeof person); // Ausgabe: object

Hier wird der Datentyp jeder Variablen in der Konsole ausgegeben.

Beachte, dass null als object angezeigt wird, was ein bekanntes Problem in JavaScript ist.

Zusätzlich gibt es auch spezielle Datentypen wie Symbol und BigInt, die in bestimmten Situationen nützlich sein können.

Beispiel für Symbol:

            
                const symbol1 = Symbol("Beschreibung");
                const symbol2 = Symbol("Beschreibung");
                console.log(symbol1 === symbol2); // Ausgabe: false

Hier wird ein Symbol mit einer Beschreibung erstellt. Beachte, dass zwei Symbole mit derselben Beschreibung nicht gleich sind.

Beispiel für BigInt:

            
                const bigInt = 1234567890123456789012345678901234567890n;
                console.log(bigInt); // Ausgabe: 1234567890123456789012345678901234567890n

Hier wird eine große Zahl mit BigInt erstellt. Beachte das Suffix n, das angibt, dass es sich um einen BigInt-Wert handelt.

Zusammenfassend ist es wichtig, die verschiedenen Datentypen in JavaScript zu verstehen, um effektiv mit Variablen und Werten arbeiten zu können. Du kannst den Datentyp einer Variablen mit dem Operator typeof überprüfen.

📖 3.3 Operatoren

Operatoren sind Symbole, die eine Operation auf Variablen oder Werten durchführen. In JavaScript gibt es verschiedene Arten von Operatoren:

  • Zuweisungsoperatoren: Zuweisung von Werten an Variablen, z.B. =
  • Mathematische Operatoren: Durchführung mathematischer Berechnungen, z.B. +, -, *, /
  • Vergleichsoperatoren: Vergleich von Werten, z.B. ==, ===, >
  • Verkettungsoperator: Verknüpfung von Strings, z.B. +

Beispiel:

            
                let a = 10;
                let b = 5;
                let summe = a + b; // 15
                console.log("Die Summe ist: " + summe);

Hier wird die Summe von a und b berechnet und in der Konsole ausgegeben.

Beispiel für Vergleichsoperatoren:

            
                let x = 10;
                let y = 20;
                console.log(x > y); // false
                console.log(x < y); // true
                console.log(x == 10); // true
                console.log(x === "10"); // false

Hier werden verschiedene Vergleichsoperatoren verwendet, um Werte zu vergleichen und die Ergebnisse in der Konsole auszugeben.

Beispiel für Verkettungsoperator:

Der Verkettungsoperator + wird verwendet, um Strings miteinander zu verbinden.

Beispiel:

            
                let vorname = "Anna";
                let nachname = "Müller";
                let vollerName = vorname + " " + nachname;
                console.log(vollerName); // Anna Müller

Hier werden die Variablen vorname und nachname mit einem Leerzeichen dazwischen zu einem vollständigen Namen verkettet.

Beispiel für Zuweisungsoperator:

Der Zuweisungsoperator = wird verwendet, um einer Variablen einen Wert zuzuweisen.

Beispiel:

            
                let alter = 15; // Zuweisung
                console.log(alter); // Ausgabe: 15

Hier wird der Wert 15 der Variablen alter zuzuweisen und anschließend in der Konsole ausgegeben.

Beispiel für mathematische Operatoren:

Mathematische Operatoren werden verwendet, um Berechnungen durchzuführen.

Beispiel:

            
                let a = 10;
                let b = 5;
                let summe = a + b; // 15
                let differenz = a - b; // 5
                let produkt = a * b; // 50
                let quotient = a / b; // 2
                console.log("Summe:", summe);
                console.log("Differenz:", differenz);
                console.log("Produkt:", produkt);
                console.log("Quotient:", quotient);

📖 3.4 Konsole

Die Konsole ist ein wichtiges Werkzeug in der Webentwicklung. Sie ermöglicht es dir, Ausgaben deines Codes anzuzeigen und Fehler zu debuggen. In JavaScript kannst du die Konsole mit dem Befehl console.log() verwenden.

Beispiel:

            
                console.log("Hallo, Welt!"); // Ausgabe: Hallo, Welt!

Hier wird der Text "Hallo, Welt!" in der Konsole ausgegeben.

Du kannst auch Variablen in der Konsole ausgeben:

            
                let name = "Max";
                console.log(name); // Ausgabe: Max

Hier wird der Wert der Variablen name in der Konsole ausgegeben.

Um die Konsole zu öffnen, kannst du in deinem Browser mit der rechten Maustaste auf die Seite klicken und "Untersuchen" oder "Element untersuchen" auswählen. Dann gehst du zum Tab "Konsole".

In der Konsole kannst du auch direkt JavaScript-Code eingeben und ausführen. Probiere es aus!

📝 4 Übungsaufgaben - Theorie

🟢 Aufgabe 4.1

Bestimme, welche der folgenden Optionen gültige Variablennamen in JavaScript sind. Kreuze die richtigen Antworten an.

Lösung anzeigen

Lösung:

  • 123variable: Falsch, da Variablennamen nicht mit einer Zahl beginnen dürfen.
  • meine-variable: Falsch, da Bindestriche in Variablennamen nicht erlaubt sind.
  • meineVariable: Richtig, da dieser Name den Regeln für Variablennamen entspricht.
  • var: Falsch, da es sich um ein reserviertes Schlüsselwort handelt.
🟢 Aufgabe 4.2

Bestimme den Datentyp, den die Variable x nach der Zuweisung hat.

                
                    let x = "Hallo Welt";
Lösung anzeigen

Lösung:

  • Number: Falsch, da x ein String ist.
  • String: Richtig, da x den Wert "Hallo Welt" hat.
  • Boolean: Falsch, da x kein Wahrheitswert ist.
  • undefined: Falsch, da x einen definierten Wert hat.
🟢 Aufgabe 4.3

Benenne den Operator, um zwei oder mehr Werte zu addieren.

Lösung anzeigen

Lösung:

  • -: Falsch, da dieser Operator für Subtraktion verwendet wird.
  • *: Falsch, da dieser Operator für Multiplikation verwendet wird.
  • +: Richtig, da dieser Operator für die Addition von Werten verwendet wird.
  • /: Falsch, da dieser Operator für Division verwendet wird.
🔵 Aufgabe 4.4

Erkläre den Unterschied zwischen den Datentypen String und Number in JavaScript.

Lösung anzeigen

Lösung:

String steht für Zeichenwerte und Number für Zahlenwerte. Bei String werden dann Zahlen verknüpft und bei Number berechnet.

🔵 Aufgabe 4.5

Beschreibe die Funktion des Zuweisungsoperators = und des Verkettungsoperators +.

Lösung anzeigen

Lösung:

Die Funktion des Zuweisungsoperators (=) besteht darin, einer Variablen einen Wert zuzuweisen. Der Verkettungsoperator (+) wird verwendet, um Zeichenketten (Strings) miteinander zu verbinden.

🔴 Aufgabe 4.6

Erläutere die Unterschiede zwischen let und const.

Lösung anzeigen

Lösung:

Der let-Operator wird verwendet, um Variablen zu deklarieren, die ihren Wert ändern können. Der const-Operator wird verwendet, um Konstanten zu deklarieren, deren Wert nicht geändert werden kann.

🔴 Aufgabe 4.7

Erläutere die Ausgabe, die der Code erzeugt.

                        
                            let age = 15;
                            let isAdult = age >= 18;
                            console.log("Volljährig?", isAdult);
Lösung anzeigen

Lösung:

Es wird "false" ausgegeben, da age < 18.

Hier ist ein vollständiges Beispiel, das zeigt, wie du Variablen, Datentypen und Operatoren in JavaScript verwenden kannst, um Berechnungen durchzuführen und Ergebnisse in der Konsole auszugeben.

HTML-Code:

                
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">

                    <body>
                        <h1>Berechnungen mit JavaScript</h1>

                    // Variablen deklarieren
                    let zahl1 = 10;
