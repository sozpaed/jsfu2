ℹ️ 3 Informationen

📖 3.1 HTML-Formulare

HTML-Formulare ermöglichen es Benutzern, Daten einzugeben und an den Server oder JavaScript zu übermitteln. Ein einfaches Formular könnte so aussehen:

            
                <form id="meinFormular">
                    <label for="name">Name:</label>
                    <input type="text" id="name" name="name">
                    <button type="submit">Absenden</button>
                </form>

Das Formular enthält ein Textfeld für den Namen und einen Absende-Button.

Hier ist eine kurze Erklärung der wichtigsten Attribute:

  • id: Eindeutige Kennung des Formulars, die in JavaScript verwendet wird.
  • name: Name des Eingabefelds, das beim Absenden des Formulars gesendet wird.
  • type: Typ des Eingabefelds (z.B. text, email, password, etc.).
  • value: Der Standardwert des Eingabefelds.
  • placeholder: Ein Platzhaltertext, der im Eingabefeld angezeigt wird, wenn es leer ist.
  • required: Gibt an, dass das Eingabefeld ausgefüllt werden muss, bevor das Formular gesendet werden kann.
  • disabled: Deaktiviert das Eingabefeld, sodass der Benutzer keine Eingaben vornehmen kann.
  • readonly: Das Eingabefeld ist schreibgeschützt, der Benutzer kann den Wert nicht ändern.
  • maxlength: Gibt die maximale Anzahl von Zeichen an, die in das Eingabefeld eingegeben werden können.
  • min: Gibt den minimalen Wert für numerische Eingaben an.
  • max: Gibt den maximalen Wert für numerische Eingaben an.
  • pattern: Ein regulärer Ausdruck, der angibt, welches Format die Eingabe haben muss.
  • autocomplete: Gibt an, ob der Browser Vorschläge für die Eingabe machen soll.
  • autofocus: Das Eingabefeld erhält automatisch den Fokus, wenn die Seite geladen wird.
  • multiple: Erlaubt die Auswahl mehrerer Dateien in einem Datei-Upload-Feld.
  • accept: Gibt an, welche Dateitypen in einem Datei-Upload-Feld akzeptiert werden.
  • action: Die URL, an die das Formular gesendet wird.
  • method: Die HTTP-Methode, die beim Senden des Formulars verwendet wird (z.B. GET, POST).
  • enctype: Der Kodierungsstil, der beim Senden des Formulars verwendet wird (z.B. application/x-www-form-urlencoded, multipart/form-data, text/plain).
  • target: Gibt an, wo die Antwort des Servers angezeigt werden soll (z.B. _self, _blank, _parent, _top).
  • novalidate: Verhindert die Standardvalidierung des Browsers beim Absenden des Formulars.
  • onsubmit: Ein JavaScript-Ereignis, das ausgelöst wird, wenn das Formular übermittelt wird.
  • onreset: Ein JavaScript-Ereignis, das ausgelöst wird, wenn das Formular zurückgesetzt wird.
  • onsubmit: Ein JavaScript-Ereignis, das ausgelöst wird, wenn das Formular übermittelt wird.
  • onreset: Ein JavaScript-Ereignis, das ausgelöst wird, wenn das Formular zurückgesetzt wird.

                    <br>
                    <label for="alter">Alter:</label>
                    <input type="number" id="alter" name="alter" min="0" max="120">

📖 3.2 Formularelemente mit JavaScript auswählen

Mit JavaScript kannst du Formularelemente auswählen und deren Werte abrufen. Beispiel:

            
                let nameFeld = document.getElementById("name");
                console.log(nameFeld.value); // Gibt den eingegebenen Namen aus

Hier wird der Wert des Eingabefelds mit der ID name abgerufen.

Du kannst auch andere Methoden verwenden, um Formularelemente auszuwählen:

  • document.querySelector("#meinFormular input[name='name']"): Wählt das Eingabefeld mit dem Namen "name" im Formular mit der ID "meinFormular" aus.
  • document.querySelectorAll("input[type='text']"): Wählt alle Text-Eingabefelder auf der Seite aus.
  • document.forms["meinFormular"].elements["name"]: Wählt das Eingabefeld mit dem Namen "name" im Formular mit der ID "meinFormular" aus.
  • document.getElementsByName("name"): Wählt alle Elemente mit dem Namen "name" aus.
  • document.getElementsByClassName("meinInput"): Wählt alle Elemente mit der Klasse "meinInput" aus.
  • document.getElementsByTagName("input"): Wählt alle Eingabefelder auf der Seite aus.
  • document.querySelector("#meinFormular"): Wählt das gesamte Formular mit der ID "meinFormular" aus.
  • document.querySelectorAll("#meinFormular input"): Wählt alle Eingabefelder im Formular mit der ID "meinFormular" aus.
  • document.querySelector("#meinFormular select"): Wählt das Dropdown-Menü im Formular mit der ID "meinFormular" aus.
  • document.querySelector("#meinFormular textarea"): Wählt das Textbereich im Formular mit der ID "meinFormular" aus.
  • document.querySelector("#meinFormular button"): Wählt den Absende-Button im Formular mit der ID "meinFormular" aus.
  • document.querySelector("#meinFormular input[type='checkbox']"): Wählt das Kontrollkästchen im Formular mit der ID "meinFormular" aus.
  • document.querySelector("#meinFormular input[type='radio']"): Wählt das Optionsfeld im Formular mit der ID "meinFormular" aus.
  • document.querySelector("#meinFormular input[type='file']"): Wählt das Datei-Upload-Feld im Formular mit der ID "meinFormular" aus.
  • document.querySelector("#meinFormular input[type='color']"): Wählt das Farbauswahlfeld im Formular mit der ID "meinFormular" aus.
  • document.querySelector("#meinFormular input[type='date']"): Wählt das Datumsauswahlfeld im Formular mit der ID "meinFormular" aus.
  • document.querySelector("#meinFormular input[type='time']"): Wählt das Zeit-Auswahlfeld im Formular mit der ID "meinFormular" aus.
  • document.querySelector("#meinFormular input[type='search']"): Wählt das Suchfeld im Formular mit der ID "meinFormular" aus.
  • document.querySelector("#meinFormular input[type='password']"): Wählt das Passwortfeld im Formular mit der ID "meinFormular" aus.
  • document.querySelector("#meinFormular input[type='submit']"): Wählt den Absende-Button im Formular mit der ID "meinFormular" aus.
  • document.querySelector("#meinFormular input[type='reset']"): Wählt den Zurücksetzen-Button im Formular mit der ID "meinFormular" aus.
  • document.querySelector("#meinFormular input[type='button']"): Wählt einen Button im Formular mit der ID "meinFormular" aus.

Hier ist ein Beispiel, wie du den Wert eines Eingabefelds abrufen kannst:

            
                let formular = document.getElementById("meinFormular");
                let nameFeld = formular.elements["name"];
                console.log(nameFeld.value); // Gibt den eingegebenen Namen aus

Hier wird der Wert des Eingabefelds mit dem Namen "name" im Formular mit der ID "meinFormular" abgerufen.

Du kannst auch den Wert eines Dropdown-Menüs abrufen:

            
                let formular = document.getElementById("meinFormular");
                let landFeld = formular.elements["land"];
                console.log(landFeld.value); // Gibt das ausgewählte Land aus

Hier wird der Wert des Dropdown-Menüs mit dem Namen "land" im Formular mit der ID "meinFormular" abgerufen.

Du kannst auch den Wert eines Kontrollkästchens abrufen:

            
                let formular = document.getElementById("meinFormular");
                let newsletterFeld = formular.elements["newsletter"];
                console.log(newsletterFeld.checked); // Gibt true oder false aus

Hier wird der Wert des Kontrollkästchens mit dem Namen "newsletter" im Formular mit der ID "meinFormular" abgerufen. Wenn das Kontrollkästchen aktiviert ist, gibt es true zurück, andernfalls false.

Du kannst auch den Wert eines Optionsfelds abrufen:

            
                let formular = document.getElementById("meinFormular");
                let geschlechtFeld = formular.elements["geschlecht"];
                for (let i = 0; i < geschlechtFeld.length; i++) {
                    if (geschlechtFeld[i].checked) {
                        console.log(geschlechtFeld[i].value); // Gibt das ausgewählte Geschlecht aus
                    }
                }

Hier wird der Wert des Optionsfelds mit dem Namen "geschlecht" im Formular mit der ID "meinFormular" abgerufen. Wenn das Optionsfeld aktiviert ist, gibt es den Wert des ausgewählten Geschlechts aus.

Hier ist ein komplettes Beispiel, wie du mehrere Eingabefelder in einem Formular mit JavaScript verarbeiten kannst:

            
            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Formular mit mehreren Feldern</title>
                <style>
                body {
                    font-family: Arial, sans-serif;
                    margin: 20px;
                }
                form {
                    max-width: 400px;
                    margin: 0 auto;
                }
                label, input, button {
                    display: block;
                    width: 100%;
                    margin-bottom: 10px;
                }
                button {
                    background-color: #007BFF;
                    color: white;
                    border: none;
                    padding: 10px;
                    cursor: pointer;
                }
                button:hover {
                    background-color: #0056b3;
                }
                .message {
                    margin-top: 20px;
                    font-size: 16px;
                    color: red;
                }
                </style>
            </head>
            <body>
                <h1>Formular mit mehreren Feldern</h1>
                <form id="meinFormular">
                <label for="vorname">Vorname:</label>
                <input type="text" id="vorname" name="vorname" required>
                <label for="nachname">Nachname:</label>
                <input type="text" id="nachname" name="nachname" required>
                <label for="email">E-Mail:</label>
                <input type="email" id="email" name="email" required>
                <button type="submit">Absenden</button>
                </form>
                <p class="message" id="meldung"></p>
                <script>
                document.getElementById("meinFormular").addEventListener("submit", function(event) {
                    event.preventDefault(); // Verhindert die Standardaktion
                    let vorname = document.getElementById("vorname").value.trim();
                    let nachname = document.getElementById("nachname").value.trim();
                    let email = document.getElementById("email").value.trim();
                    let meldung = document.getElementById("meldung");

                    if (!vorname || !nachname || !email) {
                    meldung.textContent = "Bitte füllen Sie alle Felder aus.";
                    meldung.style.color = "red";
                    } else {
                    meldung.textContent = "Formular übermittelt mit: Vorname: " + vorname + ", Nachname: " + nachname + ", E-Mail: " + email;
                    meldung.style.color = "green";
                    }
                });
                </script>
            </body>
            </html>

📖 3.3 Auf Formularübermittlungen reagieren

Du kannst auf die Übermittlung eines Formulars reagieren, indem du ein Ereignis-Listener hinzufügst. Beispiel:

            
                let formular = document.getElementById("meinFormular");
                formular.addEventListener("submit", function(event) {
                    event.preventDefault(); // Verhindert die Standardaktion (Seitenneuladen)
                    console.log("Formular wurde übermittelt!");
                });

Hier wird die Standardaktion des Formulars verhindert und eine Nachricht in der Konsole ausgegeben.

Du kannst auch auf andere Ereignisse reagieren, z.B. wenn der Benutzer ein Eingabefeld verlässt:

            
                let nameFeld = document.getElementById("name");
                nameFeld.addEventListener("blur", function() {
                    console.log("Eingabefeld wurde verlassen!");
                });

Hier wird eine Nachricht in der Konsole ausgegeben, wenn der Benutzer das Eingabefeld mit der ID name verlässt.

Du kannst auch auf andere Ereignisse reagieren, z.B. wenn der Benutzer ein Eingabefeld ändert:

            
                let nameFeld = document.getElementById("name");
                nameFeld.addEventListener("input", function() {
                    console.log("Eingabefeld wurde geändert!");
                });

Hier wird eine Nachricht in der Konsole ausgegeben, wenn der Benutzer das Eingabefeld mit der ID name ändert.

Hier ist ein komplettes Beispiel, wie du auf die Übermittlung eines Formulars reagieren kannst:

            <html lang="en">
            <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Formularübermittlung</title>
            <style>
                body {
                font-family: Arial, sans-serif;
                margin: 20px;
                }
                form {
                max-width: 300px;
                margin: 0 auto;
                }
                label, input, button {
                display: block;
                width: 100%;
                margin-bottom: 10px;
                }
                button {
                background-color: #007BFF;
                color: white;
                border: none;
                padding: 10px;
                cursor: pointer;
                }
                button:hover {
                background-color: #0056b3;
                }
                .message {
                margin-top: 20px;
                font-size: 16px;
                color: red;
                }
            </style>
            </head>
            <body>
            <h1>Formularübermittlung</h1>
            <form id="meinFormular">
                <label for="name">Name:</label>
                <input type="text" id="name" name="name" required>
                <button type="submit">Absenden</button>

                document.getElementById("meinFormular").addEventListener("submit", function(event) {
                event.preventDefault(); // Verhindert die Standardaktion
                let name = document.getElementById("name").value.trim();
                let meldung = document.getElementById("meldung");
                if (name === "") {
                    meldung.textContent = "Bitte geben Sie einen Namen ein.";
                    meldung.style.color = "red";
                } else {
                    meldung.textContent = "Formular übermittelt mit Name: " + name;
                    meldung.style.color = "green";
                }
                });
            </script>
            </body>
            </html>

📖 3.4 jQuery

jQuery ist eine JavaScript-Bibliothek, die es einfacher macht, HTML-Elemente zu manipulieren, Ereignisse zu behandeln und Animationen zu erstellen. In den vorherigen Kapiteln hast du gelernt, wie du mit reinem JavaScript:

  • HTML-Formulare erstellst und deren Elemente auswählst,
  • Werte aus Formularelementen abrufst,
  • und auf Formularübermittlungen reagierst.

Mit jQuery kannst du diese Aufgaben mit weniger Code und auf eine intuitivere Weise umsetzen.

🔹 Verknüpfung von jQuery

Um jQuery zu verwenden, musst du die jQuery-Bibliothek in dein Projekt einbinden. Dies kannst du tun, indem du das folgende <script>-Tag in den <head>- oder <body>-Bereich deiner HTML-Datei einfügst:

            
                <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

🔹 Auswahl von Formularelementen

In Kapitel 3.1 hast du gelernt, wie du Formularelemente mit document.getElementById() auswählst. Mit jQuery kannst du dies einfacher umsetzen:

            
                // Auswahl eines Eingabefelds mit der ID "name"
                let nameFeld = $("#name");

Hier wird das Eingabefeld mit der ID name ausgewählt. Du kannst direkt mit dem ausgewählten Element arbeiten.

🔹 Werte aus Formularelementen abrufen

In Kapitel 3.2 hast du gelernt, wie du Werte aus Formularelementen abrufst. Mit jQuery kannst du dies einfacher umsetzen:

            
                // Wert eines Eingabefelds abrufen
                let name = $("#name").val();
                console.log("Eingegebener Name:", name);

🔹 Auf Formularübermittlungen reagieren

In Kapitel 3.3 hast du gelernt, wie du auf die Übermittlung eines Formulars reagierst. Mit jQuery kannst du dies einfacher umsetzen:

            
                // Formularübermittlung behandeln
                $("#meinFormular").submit(function(event) {
                    event.preventDefault(); // Verhindert die Standardaktion
                    let name = $("#name").val();
                    if (name.trim() === "") {
                        console.log("Bitte geben Sie einen Namen ein.");
                    } else {
                        console.log("Formular übermittelt mit Name:", name);
                    }
                });

                            // 4. Das Formular mit document.getElementById() auswählen

🛠️ 9 Beispiel

Hier ist ein vollständiges Beispiel, das zeigt, wie du mit HTML, CSS und JavaScript ein Formular erstellen und verarbeiten kannst. In diesem Beispiel wird ein Formular verwendet, um den Namen des Benutzers abzurufen und eine Nachricht anzuzeigen.

HTML-Code:

                
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>Formular Beispiel</title>
                        <link rel="stylesheet" href="styles.css">
                        <script src="script.js"></script>
                    </head>
                    <body>
                        <h1>Formular Beispiel</h1>
                        <form id="meinFormular">
                            <label for="name">Name:</label>
                            <input type="text" id="name" name="name" required>
                            <button type="submit">Absenden</button>
                        </form>
                        <p id="meldung"></p>
                    </body>
                    </html>

CSS-Code (styles.css):

                
                    body {
                        font-family: Arial, sans-serif;
                        text-align: center;
                        margin-top: 50px;
                    }
    
                    form {
                        margin: 20px auto;
                        width: 300px;
                    }
    
                    input, button {
                        display: block;
                        width: 100%;
                        margin: 10px 0;
                        padding: 10px;
                        font-size: 16px;
                    }
    
                    #meldung {
                        margin-top: 20px;
                        font-size: 18px;
                        color: green;
                    }

JavaScript-Code (script.js):

                
                    document.getElementById("meinFormular").addEventListener("submit", function(event) {
                        event.preventDefault(); // Verhindert die Standardaktion
                        let name = document.getElementById("name").value.trim();
                        let meldung = document.getElementById("meldung");
                        if (name === "") {
                            meldung.textContent = "Bitte geben Sie einen Namen ein.";
                            meldung.style.color = "red";
                        } else {
                            meldung.textContent = "Hallo, " + name + "!";
                            meldung.style.color = "green";
                        }
                    });
