ℹ️ 3 Informationen

📖 3.1 HTML-Formulare

HTML-Formulare ermöglichen es Benutzern, Daten einzugeben und an den Server oder JavaScript zu übermitteln. Ein einfaches Formular könnte so aussehen:

            
                <form id="meinFormular">
                    <label for="name">Name:</label>
                    <input type="text" id="name" name="name">
                    <button type="submit">Absenden</button>
                </form>

Das Formular enthält ein Textfeld für den Namen und einen Absende-Button.

📖 3.2 Formularelemente mit JavaScript auswählen

Mit JavaScript kannst du Formularelemente auswählen und deren Werte abrufen. Beispiel:

            
                let nameFeld = document.getElementById("name");
                console.log(nameFeld.value); // Gibt den eingegebenen Namen aus

Hier wird der Wert des Eingabefelds mit der ID name abgerufen.

📖 3.3 Auf Formularübermittlungen reagieren

Du kannst auf die Übermittlung eines Formulars reagieren, indem du ein Ereignis-Listener hinzufügst. Beispiel:

            
                let formular = document.getElementById("meinFormular");
                formular.addEventListener("submit", function(event) {
                    event.preventDefault(); // Verhindert die Standardaktion (Seitenneuladen)
                    console.log("Formular wurde übermittelt!");
                });

Hier wird die Standardaktion des Formulars verhindert und eine Nachricht in der Konsole ausgegeben.
