📙 JS :: Modul 3

🎬 1 Teaser

In diesem Modul lernst du, wie du mit JavaScript die Gestaltung von HTML-Elementen dynamisch ändern und einfache Berechnungen durchführen kannst. Du wirst verstehen, wie du CSS-Klassen manipulierst und Berechnungsergebnisse auf der Webseite anzeigst.

🎯 2 Lernziele

  • ✅ Die Inline-Stile von HTML-Elementen ändern können.
  • ✅ CSS-Klassen mit JavaScript hinzufügen, entfernen und umschalten können.
  • ✅ Numerische Eingaben aus HTML-Elementen abrufen können.
  • ✅ Grundlegende arithmetische Berechnungen mit JavaScript durchführen können.
  • ✅ Die Ergebnisse von Berechnungen auf der Webseite anzeigen können.

ℹ️ 3 Informationen

📖 3.1 Inline-Stile ändern

Mit JavaScript kannst du die Inline-Stile eines HTML-Elements ändern. Beispiel:

            
                let element = document.getElementById("meinElement");
                element.style.color = "red";
                element.style.backgroundColor = "yellow";

Hier wird die Textfarbe auf Rot und der Hintergrund auf Gelb gesetzt.

📖 3.2 CSS-Klassen manipulieren

Mit JavaScript kannst du CSS-Klassen zu HTML-Elementen hinzufügen, entfernen oder umschalten. Beispiel:

            
                let element = document.getElementById("meinElement");
                element.classList.add("meineKlasse"); // Klasse hinzufügen
                element.classList.remove("meineKlasse"); // Klasse entfernen
                element.classList.toggle("meineKlasse"); // Klasse umschalten

Dies ermöglicht dir, die Gestaltung eines Elements dynamisch zu ändern.

📖 3.3 Numerische Eingaben und Berechnungen

Du kannst numerische Eingaben aus HTML-Elementen abrufen und Berechnungen durchführen. Beispiel:

            
                let zahl1 = parseFloat(document.getElementById("zahl1").value);
                let zahl2 = parseFloat(document.getElementById("zahl2").value);
                let summe = zahl1 + zahl2;
                document.getElementById("ergebnis").innerText = "Summe: " + summe;

Hier werden zwei Zahlen aus Eingabefeldern abgerufen, addiert und das Ergebnis angezeigt.
