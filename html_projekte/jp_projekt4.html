🔢 JS :: Projekt 4 :: Zahlenraten

🎬 1 Teaser

In diesem Projekt lernst du, wie du ein Zahlenratenspiel mit JavaScript umsetzen kannst. Der Computer wählt eine zufällige Zahl, und du musst sie erraten.

Das Projekt ist in Module aufgeteilt, die du Schritt für Schritt bearbeiten kannst. Am Ende setzt du alle Teile zusammen, um das vollständige Spiel zu erstellen.

Füge jeweils pro Modul den JavaScript-Code (jp_zahlenraten.js) und den HTML-Code (jp_zahlenraten.html) ein. Schritt für Schritt näherst du dich dann dem Programm.

Verwende den CSS-Code, der am Ende der Seite angegeben wird, und lege diesen in der Datei `jp_projekte.css` ab.

🛠️ 2 Kriterien

Die Aufteilung des Programms richtet sich nach den Modulen:

  • Modul 1: Grundlagen von JavaScript
  • Modul 2: Interaktion mit der Webseite
  • Modul 3: Dynamische Gestaltung und Berechnungen
  • Modul 4: Benutzerinteraktion über Formulare
  • Modul 5: Entwicklung interaktiver Elemente

Dieses Projekt ist bestanden, wenn alle der folgenden Punkte erfüllt sind:

  • Das Projekt ist vollständig und funktionsfähig.
  • Der HTML-, JavaScript- und CSS-Code sind jeweils ausführlich kommentiert.
  • Die HTML-Datei ist mit dem JavaScript- und CSS-Code verknüpft.

📕 Modul 4: Benutzerinteraktion über Formulare

Der folgende Code liest die Eingabe des Benutzers aus und überprüft, ob sie gültig ist.

JavaScript-Code (jp_projekt4.js)

            
                // Passe den Code aus Modul 3 an.
                let randomNumber = getRandomNumber(); // Zufallszahl zwischen 1 und 10
                let attempts = 0; // Zählt die Versuche des Nutzers

                $('#sozStartButton').click(function () {
                    playGame();
                });
                function isValidGuess(guess) {
                    return !isNaN(guess) && guess >= 1 && guess <= 10;
                }
                function playGame() {
                    let userChoice = parseInt($('#firstNumber').val());

                    if (isNaN(userChoice) || userChoice < 1 || userChoice > 10) {
                        updateResultDisplay('Bitte nur Zahlen zwischen 1 und 10 eingeben.', 'error');
                        return;
                    }

                    attempts++;
                    let result = checkGuess(userChoice, randomNumber);

                    updateResultDisplay(result, result.includes('gewonnen') ? 'success' : 'error');

                    if (result.includes('gewonnen')) {
                        randomNumber = getRandomNumber();
                        attempts = 0;
                    }
                }
                function checkGuess(userChoice, randomNumber) {
                    if (userChoice === randomNumber) {
                        return `Herzlichen Glückwunsch! Du hast gewonnen! Die Zahl war ${randomNumber}.`;
                    } else if (userChoice < randomNumber) {
                        return 'Deine Zahl ist zu niedrig. Versuche es erneut.';
                    } else {
                        return 'Deine Zahl ist zu hoch. Versuche es erneut.';
                    }
                }
                function updateResultDisplay(message, type) {
                    const resultElement = $('#showCallbackOfChallenge');
                    resultElement.css('visibility', 'visible');

                    if (type === 'error') {
                        resultElement.css('background-color', '#c22b67');
                    } else if (type === 'success') {
                        resultElement.css('background-color', '#28a745');
                    }

                    resultElement.html(message);
                }

Hier wird die Funktion `playGame()` definiert, die die Eingabe des Benutzers überprüft und das Ergebnis anzeigt.

Die Funktion `isValidGuess()` überprüft, ob die Eingabe des Benutzers eine gültige Zahl zwischen 1 und 10 ist.

Die Funktion `checkGuess()` überprüft, ob die Eingabe des Benutzers mit der Zufallszahl übereinstimmt und gibt das Ergebnis zurück.

Die Funktion `updateResultDisplay()` aktualisiert die Anzeige des Ergebnisses auf der Webseite.

Die Funktion `playGame()` wird aufgerufen, wenn der Button "Raten" geklickt wird.

Die Funktion `getRandomNumber()` wird aufgerufen, um eine neue Zufallszahl zu generieren, wenn der Benutzer gewonnen hat.
