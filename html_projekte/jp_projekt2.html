📙 Modul 3: Dynamische Gestaltung und Berechnungen

In diesem Kapitel lernst du, wie du dynamische Inhalte erstellst und Berechnungen durchführst. Der folgende Code zeigt, wie der größte gemeinsame Teiler (ggT) berechnet wird.

            
                function calculateGCD(a, b) {
                    while (b !== 0) {
                        let temp = b;
                        b = a % b;
                        a = temp;
                    }
                    return a;
                }

📕 Modul 4: Benutzerinteraktion über Formulare

Hier lernst du, wie du Benutzereingaben aus Formularen verarbeitest. Der folgende Code liest die Werte aus den Eingabefeldern aus und überprüft, ob sie gültig sind.

            
                function getInputValues() {
                    let firstNumber = $('#firstNumber').val();
                    let secondNumber = $('#secondNumber').val();
                    firstNumber = parseInt(firstNumber);
                    secondNumber = parseInt(secondNumber);

                    if (isNaN(firstNumber) || isNaN(secondNumber)) {
                        alert('Bitte gültige Zahlen eingeben.');
                        return null;
                    }
                    return { firstNumber, secondNumber };
                }

                function updateResultDisplay(message, type) {
                    const resultElement = $('#showCallbackOfChallenge');
                    resultElement.css('visibility', 'visible');

                    if (type === 'error') {
                        resultElement.css('background-color', '#c22b67');
                    } else if (type === 'success') {
                        resultElement.css('background-color', '#28a745');
                    }

                    resultElement.html(message);
                }
