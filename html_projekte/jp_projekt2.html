📙 Modul 3: Dynamische Gestaltung und Berechnungen

Der folgende Code zeigt, wie der größte gemeinsame Teiler (ggT) berechnet wird.

JavaScript-Code (jp_ggt.js)

            
                // Tausche den Code aus Modul 2 mit dem unteren Code.
                $(document).ready(function () {
                    $('#sozStartButton').click(function () {
                        let firstNumber = $('#firstNumber').val();
                        let secondNumber = $('#secondNumber').val();
                        let ggtResult = calculateGCD(firstNumber, secondNumber);
                        $('#showCallbackOfChallenge').html("Der größte gemeinsame Teiler ist " + ggtResult);
                        $('#showCallbackOfChallenge').css('visibility', 'visible');
                        $('#showCallbackOfChallenge').css('background-color', '#28a745');
                        $('#showCallbackOfChallenge').css('color', '#fff'); 
                    });
                });

                function calculateGCD(value1, value2) {
                    value1 = parseInt(value1, 10);
                    value2 = parseInt(value2, 10);
                
                    if (isNaN(value1) || isNaN(value2) || value1 <= 0 || value2 <= 0) {
                        return 'Bitte gültige positive Zahlen eingeben.';
                    }
                
                    while (value2 !== 0) {
                        let temp = value2;
                        value2 = value1 % value2;
                        value1 = temp;
                    }
                    return value1;
                }

Der Code berechnet den größten gemeinsamen Teiler (ggT) zweier Zahlen, die der Benutzer eingibt. Wenn die Eingaben ungültig sind, wird eine Fehlermeldung angezeigt.

Die Funktion `calculateGCD` berechnet den ggT mithilfe des Euklidischen Algorithmus. Die Funktion `updateResultDisplay` aktualisiert die Anzeige des Ergebnisses auf der Webseite.
