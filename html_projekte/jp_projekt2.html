📐 JS :: Projekt 2 :: Größter gemeinsamer Teiler

🎬 1 Teaser

In diesem Projekt kannst du den größten gemeinsamen Teiler (ggT) zweier Zahlen berechnen. Gib einfach zwei Zahlen ein und drücke auf "Berechnen".

Das Tool zeigt dir den ggT der beiden Zahlen an. Viel Erfolg!

Füge jeweils pro Modul den JavaScript-Code (jp_ggt.js) und den HTML-Code (jp_ggt.html) ein. Schritt für Schritt näherst du dich dann dem Programm.

Verwende den CSS-Code, der am Ende der Seite angegeben wird, und lege diesen in der Datei `jp_projekte.css` ab.

JavaScript-Code (jp_ggt.js)

            
                $(document).ready(function () {
                    $('#sozStartButton').click(function () {
                        calculateGGT();
                    });

                    function calculateGGT() {
                        let firstNumber = $('#firstNumber').val();
                        let secondNumber = $('#secondNumber').val();

                        firstNumber = parseInt(firstNumber);
                        secondNumber = parseInt(secondNumber);

                        if (isNaN(firstNumber) || firstNumber < 1 || isNaN(secondNumber) || secondNumber < 1) {
                            updateResultDisplay('Bitte gültige Zahlen eintragen.', 'error');
                            return;
                        }

                        let ggtResult = calculateGCD(firstNumber, secondNumber);
                        updateResultDisplay(`Der größte gemeinsame Teiler von ${firstNumber} und ${secondNumber} ist: ${ggtResult}`, 'success');
                    }

                    function calculateGCD(value1, value2) {
                        while (value2 !== 0) {
                            let temp = value2;
                            value2 = value1 % value2;
                            value1 = temp;
                        }
                        return value1;
                    }

                    function updateResultDisplay(message, type) {
                        const resultElement = $('#showCallbackOfChallenge');
                        resultElement.css('visibility', 'visible');

                        if (type === 'error') {
                            resultElement.css('background-color', '#c22b67');
                        } else if (type === 'success') {
                            resultElement.css('background-color', '#28a745');
                        }

                        resultElement.html(message);
                    }
                });
