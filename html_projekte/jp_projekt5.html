🎮 JS :: Projekt 5 :: Stein, Schere, Papier

🎬 1 Teaser

In diesem Projekt lernst du, wie du das Spiel "Stein, Schere, Papier" mit JavaScript umsetzen kannst. Du wirst verstehen, wie du Benutzereingaben verarbeitest, Zufallszahlen generierst und Spielregeln implementierst.

Das Projekt ist in Module aufgeteilt, die du Schritt für Schritt bearbeiten kannst. Am Ende setzt du alle Teile zusammen, um das vollständige Spiel zu erstellen.

Füge jeweils pro Modul den JavaScript-Code (jp_steinscherepapier.js) und den HTML-Code (jp_steinscherepapier.html) ein. Schritt für Schritt näherst du dich dann dem Programm.

Verwende den CSS-Code, der am Ende der Seite angegeben wird, und lege diesen in der Datei `jp_projekte.css` ab.

📕 Modul 4: Benutzerinteraktion über Formulare

Der folgende Code liest die Eingabe des Benutzers aus und überprüft, ob sie gültig ist.

JavaScript-Code (jp_steinscherepapier.js)

            
                // Ergänze den Code aus Modul 3
                function playGame() {
                    const userChoice = parseInt($('#firstNumber').val());
                    if (!isValidChoice(userChoice)) {
                        alert('Bitte nur Zahlen zwischen 1 und 3 eingeben.');
                        return;
                    }

                    const computerChoice = getComputerChoice();
                    alert(`Spieler: ${userChoice}. Computer: ${computerChoice}.`);
                }
                
                function determineWinner(userChoice, computerChoice) {
                    if (userChoice === computerChoice) {
                        return 'Unentschieden!';
                    } else if (
                        (userChoice === STEIN && computerChoice === SCHERE) ||
                        (userChoice === SCHERE && computerChoice === PAPIER) ||
                        (userChoice === PAPIER && computerChoice === STEIN)
                    ) {
                        return 'Gewonnen!';
                    } else {
                        return 'Verloren!';
                    }
                }
                function isValidChoice(choice) {
                    return !isNaN(choice) && choice >= 1 && choice <= 3;
                }
                
                console.log(isValidChoice(2)); // Beispiel: true

Hier wird die Funktion playGame() definiert, die die Eingabe des Benutzers überprüft und das Ergebnis des Spiels anzeigt.

Die Funktion isValidChoice() überprüft, ob die Eingabe des Benutzers eine gültige Zahl zwischen 1 und 3 ist. Wenn die Eingabe ungültig ist, wird eine Alert-Box angezeigt.

Die Funktion determineWinner() bestimmt den Gewinner des Spiels, indem sie die Wahl des Benutzers und die Wahl des Computers vergleicht. Das Ergebnis wird in einer Alert-Box angezeigt.
